## Getting started

Install Node.js 10 or above

```js
# yarn

yarn add @abhijithvijayan/tsconfig --dev

# npm

npm install --save-dev @abhijithvijayan/tsconfig
```

## Usage

`tsconfig.json`
```json
{
  "extends": "@abhijithvijayan/tsconfig",
	"compilerOptions": {
		"outDir": "dist",
		"target": "es2018" // Node.js 10
	}
}
```
